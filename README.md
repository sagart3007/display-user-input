# display-user-input
### Project overview
The aim of this project is to collect details from user i.e. User name and User age and display it.

### Key concepts used
* React
* Javascript
* HTML
* CSS
* State and Props
* useState hook


### How to run code?
To run the code, navigate into a specific code snapshot folder via the cd command in your command prompt or terminal first.
Then run npm install to install all required dependencies (this will create a /node_modules folder).
Then run npm start to launch development server
